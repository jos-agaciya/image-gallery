# image-gallery
An interactive and responsive image gallery built using HTML, CSS, and JavaScript with lightbox view, smooth transitions, and category filtering.



## ✨ Features
- Responsive grid layout  
- Next / Previous navigation  
- Lightbox (fullscreen image view)  
- Smooth hover effects and transitions  
- Category filtering (bonus feature)  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## 🚀 How to Run
1. Clone the repository  
2. Open `index.html` in your browser  
3. Explore the gallery  

---

- Event handling
